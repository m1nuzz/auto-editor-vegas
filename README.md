# FCP XML → Vegas Pro Fixer

A Python script that fixes Final Cut Pro XML files generated by [auto-editor](https://github.com/WyattBlue/auto-editor) to make them compatible with Vegas Pro.

## Features

The script applies the following fixes to ensure Vegas Pro compatibility:

1. **Format attributes** - Adds missing `fieldOrder`, `pixelAspect`, and `colorDepth` attributes
2. **Sample characteristics** - Adds missing `samplerate` (48000) and `depth` (16) elements
3. **Empty pathurl tags** - Removes empty `<pathurl>` elements from file references
4. **Link elements** - Removes `<link>` elements that cause TrackEventGroup null reference errors
5. **Audio gain reset** - Resets audio gain to 0dB and removes boosting filters
6. **Audio track cleanup** - Keeps only one audio track per sequence, removes duplicates
7. **Media structure** - Ensures `<media>` contains both `<video>` and `<audio>` blocks
8. **Clipitem fields** - Ensures required fields (`name`, `in`, `out`, `start`, `end`) are present
9. **Empty tracks** - Removes empty `<track>` elements

## Requirements

- Python 3.6+
- No external dependencies (uses only standard library)

## Usage

### Interactive Mode

```bash
python fix_for_vegas.py
```

The script will prompt you for:
- Path to the XML file (generated by auto-editor)
- Optional custom output path (default: `<filename>_vegas.xml`)

### Command Line Arguments

```bash
python fix_for_vegas.py <input_xml> [output_xml]
```

**Examples:**

```bash
# With default output path
python fix_for_vegas.py project.xml

# With custom output path
python fix_for_vegas.py project.xml fixed_project.xml
```

## Importing to Vegas Pro

After running the script:

1. Open Vegas Pro
2. Go to **File → Import → Final Cut Pro 7 XML**
3. Select the output file (e.g., `project_vegas.xml`)

## Workflow

```
auto-editor → FCP XML → fix_for_vegas.py → Vegas-compatible XML → Vegas Pro
```
